# 도시인문학 연구 대시보드

> KCI(한국학술지인용색인)에서 **'도시인문학'** 키워드로 검색된 논문 전체(447편)를 기반으로, 연구 분야의 전체적인 양상을 시각적으로 탐색하는 인터랙티브 대시보드입니다.

---

## 📌 개요

| 항목 | 내용 |
|---|---|
| 데이터 범위 | 2003–2025년, 논문 447편, 참고문헌 12,902건 |
| 핵심 학술지 | 도시인문학연구 (243편, 54%) |
| 총 인용 횟수 | 1,491회 |
| 참여 기관 수 | 137곳 |
| 수록 학술지 수 | 119종 |

---

## 🗂️ 대시보드 구조

차량 계기판처럼 **4계층 위계**로 설계되어, 전체 규모 → 시계열 흐름 → 공간·관점 지형 → 개별 영향력 순으로 탐색할 수 있습니다.

```
01  핵심 지표      KPI 5개 카드 (총 논문 수 · 연구 기간 · 인용 횟수 · 기관 수 · 학술지 수)
02  연구 흐름      연도별 논문 수 막대 차트 + 5년 단위 핵심 키워드 변화
03  연구 지형      패널 A: 지역별 연구 방향성 / 패널 B: 관점 분포 도넛 + 시기별 변화
04  인용 Top 10   영향력 기준 논문 랭킹 테이블
```

---

## ✨ 주요 기능

- **카운터 애니메이션** — 페이지 로드 시 KPI 숫자가 부드럽게 카운트업
- **인터랙티브 차트** — Chart.js 기반 막대·도넛·누적 면적 차트 (툴팁·호버 지원)
- **스크롤 진입 애니메이션** — 지역별 바와 인용 바가 뷰포트 진입 시 순차적으로 채워짐
- **지역별 개별 색상** — 8개 광역 지역에 고유 색상 배정
- **다크 테마** — 눈에 편한 딥 네이비 배경에 그라디언트 강조색
- **반응형 레이아웃** — 1100px / 700px 브레이크포인트 지원

## 🔍 주요 발견

| 시기 | 특징 | 핵심 키워드 |
|---|---|---|
| ~2010 | 이론 탐색기 | 벤야민, 도시재생, 기호학, 제3공간 |
| 2011–2015 | 개념 형성기 | 도시인문학, 공동체, 중국, 장소성 |
| 2016–2020 | 정책·제도화기 | 인문도시, 인문학 대중화, 사이보그, 공유지 |
| 2021–2025 | 이론 전환기 | 박완서, 인류세, 돌봄, 라투르, 커먼즈 |

**관점 분포:** 문화·매체(29%) · 정책·사회(25%) · 공간·장소(24%) · 철학·이론(23%)으로 4분야가 고르게 균형을 이룸

**최다 인용 논문:** 허경, 「미셸 푸코의 '헤테로토피아'」(2011) — **66회**

## 🛠️ 기술 스택

| 분류 | 내용 |
|---|---|
| 마크업 | HTML5 (시맨틱 구조) |
| 스타일 | Vanilla CSS (CSS Custom Properties, Grid, Flexbox) |
| 차트 | [Chart.js 4.4](https://www.chartjs.org/) (CDN) |
| 폰트 | [Pretendard](https://github.com/orioncactus/pretendard) · [Inter](https://fonts.google.com/specimen/Inter) (Google Fonts) |
| 애니메이션 | CSS transitions + Web Animations API (IntersectionObserver) |
| 데이터 가공 | Python 3 (openpyxl) |

별도 빌드 도구 없이 **정적 HTML 파일** 하나로 동작합니다.

## 🚀 실행 방법

별도 서버 없이 `index.html`을 브라우저에서 바로 열면 됩니다.

```bash
# macOS
open index.html

# 또는 로컬 서버 사용 (권장)
python3 -m http.server 8000
# → http://localhost:8000
```

> Chart.js와 Google Fonts를 CDN으로 불러오므로 **인터넷 연결**이 필요합니다.
